Axiom provides powerful tools to query your data. Whether you want to query logs, traces, or metrics, whether you want to use a visual interface or a text-based query language, Axiom has you covered.

Axiom has three tabs for exploring your data:
- [**Datasets**](/query-data/datasets): Browse and analyze the fields in any dataset. Use this to understand the shape of your data before writing a query.
- [**Stream**](/query-data/stream): Watch events arrive in real time and inspect individual events as they're ingested.
- **Query**: Run queries against your data using either a visual interface or a text-based query language.

The Datasets and Stream tabs work the same way for logs, traces, and metrics. The Query tab experience differs based on what you query and how you query it.

## Query tab

The Query tab is where you run queries against your data. It offers two interfaces:

- **Builder**: A visual interface for filtering, grouping, and visualizing event data.
- **Editor**: Write queries in a text-based query language.

Both Builder and Editor are available for logs and traces and for metrics, but they work differently depending on the signal type.

| | Builder (visual) | Editor (query language) |
|:---|:---|:---|
| **Logs and traces** | [Query logs and traces](/query-data/explore)<br />[Traces](/query-data/traces) for trace-specific UI | [Query logs and traces](/query-data/explore)<br />[APL query reference](/apl/introduction) |
| **Metrics** | [Query metrics using Builder](/query-data/metrics/query-metrics). | [Query metrics using MPL](/query-data/metrics/query-metrics-mpl) |

The following features are available for logs and traces:
- [Visualize](/query-data/visualizations): Charts, graphs, and other visual components to make sense of query results.
- [Views](/query-data/views): Apply filters and transformations to datasets to create reusable virtual tables with scoped access control.
- [Virtual fields](/query-data/virtual-fields): Derive new values from data in real-time during queries without pre-structuring or transforming data during ingestion.

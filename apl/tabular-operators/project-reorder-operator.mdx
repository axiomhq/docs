## Syntax

```kusto
| project-reorder FieldName1, FieldName2, ... [asc|desc]
```


## Arguments

| **name**              | **type**     | **description**                                      |
| --------------------- | ------------ | ------------------------------------------------------ |
| Field Name(Column Name) | **string**     | The name of the field to be reordered in the output. |
| [asc|desc]           | **string** |  Optional. Specifies the sort order for the reordered fields. |

## Returns

A table that has the field named 'arguments' and as many rows as the input table.

## Examples

```kusto
['http-logs']
| project ['geo.country'] = ['id']
```

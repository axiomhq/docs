Sorts the rows of the input dataset into order by one or more fields.

## Syntax

```kusto
| order by Field [asc], Field2 [desc]
```

[Run in Playground](https://play.axiom.co/axiom-play-qf1k/explorer?initForm=%7B%22apl%22%3A%22%5B%27sample-http-logs%27%5D%5Cn%7C%20order%20by%20content_type%20asc%2C%20is_tls%20desc%22%2C%22queryOptions%22%3A%7B%22quickRange%22%3A%2230d%22%7D%7D)

## Arguments

| **name** | **type**     | **description**                                                                                  |
| -------- | ------------ | ------------------------------------------------------------------------------------------------ |
| Field    | **string**     | Field of dataset by which to sort. The type of the values must be numeric, date, time, or string. |
| asc      | **expression** | Sort by into ascending order, low to high                                                        |
| desc     | **expression** | descending high to low                                                                           |

## Example

```kusto
['http-logs']
| order by ['geo.country'] asc, method desc
```

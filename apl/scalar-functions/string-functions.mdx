The table summarizes the string functions available in APL.

| Name                                       | Description                                                                                                            |
| ------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| [base64_decode_toarray](/apl/scalar-functions/string-functions/base64-decode-toarray) | Decode a Base64-encoded string into an array of bytes.                                                                                         |
| [base64_decode_tostring](/apl/scalar-functions/string-functions/base64-decode-tostring) | Decode a base64 string to a UTF-8 string.                                                                                 |
| [base64_encode_fromarray](/apl/scalar-functions/string-functions/base64-encode-fromarray) | Convert a sequence of bytes into a Base64-encoded string.                                                                                         |
| [base64_encode_tostring](/apl/scalar-functions/string-functions/base64-encode-tostring) | Encode a string as base64 string.                                                                                         |
| [coalesce](/apl/scalar-functions/string-functions/coalesce)                             | Evaluate a list of expressions and return the first non-null (or non-empty for string) expression.                       |
| [countof_regex](/apl/scalar-functions/string-functions/countof-regex)                   | Count occurrences of a substring in a string. Regex matches don’t.                                                        |
| [countof](/apl/scalar-functions/string-functions/countof)                               | Count occurrences of a substring in a string.                                                                             |
| [extract_all](/apl/scalar-functions/string-functions/extract-all)                       | Get all matches for a regular expression from a text string.                                                               |
| [extract](/apl/scalar-functions/string-functions/extract)                               | Get a match for a regular expression from a text string.                                                                   |
| [format_bytes](/apl/scalar-functions/string-functions/format-bytes)                     | Format a number of bytes as a string including bytes units                                                                |
| [format_url](/apl/scalar-functions/string-functions/format-url)                         | Format an input string into a valid URL by adding the necessary protocol if it’s escaping illegal URL characters.         |
| [gettype](/apl/scalar-functions/string-functions/gettype)                               | Return the runtime type of its single argument.                                                                           |
| [indexof](/apl/scalar-functions/string-functions/indexof)                               | Function reports the zero-based index of the first occurrence of a specified string within input string.                   |
| [isascii](/apl/scalar-functions/string-functions/isascii)                           | Checks whether all characters in an input string are ASCII characters.            |
| [isempty](/apl/scalar-functions/string-functions/isempty)                               | Return true if the argument is an empty string or is null.                                                                |
| [isnotempty](/apl/scalar-functions/string-functions/isnotempty)                         | Return true if the argument isn’t an empty string or a null.                                                              |
| [isnotnull](/apl/scalar-functions/string-functions/isnotnull)                           | Return true if the argument isn’t null.                                                                                  |
| [isnull](/apl/scalar-functions/string-functions/isnull)                                 | Evaluate its sole argument and return a bool value indicating if the argument evaluates to a null value.                 |
| [parse_bytes](/apl/scalar-functions/string-functions/parse-bytes)                       | Parse a string including byte size units and return the number of bytes                                                  |
| [parse_csv](/apl/scalar-functions/string-functions/parse-csv)                           | Split a given string representing a single record of comma-separated values and return a string array with these values. |
| [parse_json](/apl/scalar-functions/string-functions/parse-json)                         | Interpret a string as a JSON value and return the value as dynamic.                                                     |
| [parse_url](/apl/scalar-functions/string-functions/parse-url)                           | Parse an absolute URL string and return a dynamic object contains all parts of the URL.                                  |
| [parse_urlquery](/apl/scalar-functions/string-functions/parse-urlquery)                 | Parse a URL query string and return a dynamic object contains the Query parameters.                                      |
| [quote](/apl/scalar-functions/string-functions/quote)                               | Return a string representing the input enclosed in double quotes, with internal quotes and escape sequences handled appropriately.                                                                             |
| [replace_regex](/apl/scalar-functions/string-functions/replace-regex)                   | Replaces all regex matches with another string.                                                                            |
| [replace_string](/apl/scalar-functions/string-functions/replace-string)                 | Replace all string matches with another string.                                                                           |
| [replace](/apl/scalar-functions/string-functions/replace)                               | Replace all regex matches with another string.                                                                             |
| [reverse](/apl/scalar-functions/string-functions/reverse)                               | Make reverse of input string.                                                                                    |
| [split](/apl/scalar-functions/string-functions/split)                                   | Split a given string according to a given delimiter and return a string array with the contained substrings.             |
| [strcat_delim](/apl/scalar-functions/string-functions/strcat-delim)                     | Concatenate between 2 and 64 arguments, with delimiter, provided as first argument.                                       |
| [strcat](/apl/scalar-functions/string-functions/strcat)                                 | Concatenate between 1 and 64 arguments.                                                                                   |
| [strcmp](/apl/scalar-functions/string-functions/strcmp)                                 | Compare two strings.                                                                                                      |
| [string-size](/apl/scalar-functions/string-functions/string-size)                       | Return the length, in characters, of the input string.                                                                    |
| [strlen](/apl/scalar-functions/string-functions/strlen)                                 | Return the length, in characters, of the input string.                                                                    |
| [strrep](/apl/scalar-functions/string-functions/strrep)                                 | Repeat given string provided number of times (default = 1).                                                               |
| [substring](/apl/scalar-functions/string-functions/substring)                           | Extract a substring from a source string.                                                                                 |
| [tolower](/apl/scalar-functions/string-functions/tolower)                               | Convert a string to lower case.                                                                                           |
| [totitle](/apl/scalar-functions/string-functions/totitle)                               | Convert a string to title case.                                                                                           |
| [toupper](/apl/scalar-functions/string-functions/toupper)                               | Convert a string to upper case.                                                                                           |
| [translate](/apl/scalar-functions/string-functions/translate)                                     | Substitutes characters in a string, one by one, based on their position in two input lists.                                                          |
| [trim_end_regex](/apl/scalar-functions/string-functions/trim-end-regex)                 | Remove trailing match of the specified regular expression.                                                                |
| [trim_end](/apl/scalar-functions/string-functions/trim-end)                             | Remove trailing match of the specified cutset.                                                                            |
| [trim_regex](/apl/scalar-functions/string-functions/trim-regex)                         | Remove all leading and trailing matches of the specified regular expression.                                              |
| [trim_space](/apl/scalar-functions/string-functions/trim-space)                         | Remove all leading and trailing whitespace from a string.                                                                             |
| [trim_start_regex](/apl/scalar-functions/string-functions/trim-start-regex)             | Remove leading match of the specified regular expression.                                                                 |
| [trim_start](/apl/scalar-functions/string-functions/trim-start)                         | Remove leading match of the specified cutset.                                                                             |
| [trim](/apl/scalar-functions/string-functions/trim)                                     | Remove all leading and trailing matches of the specified cutset.                                                          |
| [unicode_codepoints_from_string](/apl/scalar-functions/string-functions/unicode-codepoints-from-string)                           | Convert a UTF-8 string into an array of Unicode code points. |
| [unicode_codepoints_to_string](/apl/scalar-functions/string-functions/unicode-codepoints-to-string)                           | Convert an array of Unicode code points into a UTF-8 encoded string. |
| [url_decode](/apl/scalar-functions/string-functions/url-decode)                         | Convert encoded URL into a regular URL representation.                                                       |
| [url_encode](/apl/scalar-functions/string-functions/url-encode)                         | Convert characters of the input URL into a format that can be transmitted over the Internet.                 |

Use pair functions to create, parse, and search key-value pair strings. These functions are useful for working with tags, labels, metadata, and any data stored in key-value format.

## List of functions

| Function | Description |
| -------- | ----------- |
| [find_pair](/apl/scalar-functions/pair-functions/find-pair) | Searches an array of key-value pairs and finds the first pair that matches specified key and value patterns. |
| [pair](/apl/scalar-functions/pair-functions/pair) | Creates a pair string from a key and value with a configurable separator. |
| [parse_pair](/apl/scalar-functions/pair-functions/parse-pair) | Parses a pair string into its key and value components. |

## Select time range

The time range specifies the interval for which you want to display data in the dashboard. When you change the time range, it affects the data displayed in all dashboard elements.

To select the time range:
1. In the top right, click <img src="/doc-assets/icons/clock.svg" className="inline-icon" alt="Time range" /> **Time range**.
1. Do one of the following:
    - Select a preset from the **Quick range** options.
    - Set specific times using the **Custom start/end date** fields.

## Share dashboards

To specify dashboard access:

1. In the top right, click <img src="/doc-assets/icons/share.svg" className="inline-icon" alt="Share" /> **Share**.
1. Select one of the following:
    - **Just Me** - Only you can access the dashboard.
    - A group in your Axiom organization - Only members of the selected group can access the dashboard. For more information about groups, see [Access](/reference/settings#access-overview).
    - **Everyone** - All users in your Axiom organization can access the dashboard.
1. Click **Save**.

<Note>
  Users can only see data from datasets they have access to. If a user has dashboard access but limited dataset access, they'll only see data from their authorized datasets.
</Note>

## Control display of annotations

To specify which annotations to display in dashboard elements:

1. In the top right, click <img src="/doc-assets/icons/diamond.svg" className="inline-icon" alt="Annotations" /> **Annotations**.
1. Select one of the following:
    - Show all annotations
    - Hide all annotations
    - Select specific annotation types to display
1. Click **Save**.

## Set dashboard as homepage

To set a dashboard as your browser homepage, click <img src="/doc-assets/icons/house.svg" className="inline-icon" alt="Home icon" /> **Set as homepage** in the top right.

## Enter full screen

Full-screen mode is useful for displaying the dashboard on a TV or shared monitor.

To enter full-screen mode, click <img src="/doc-assets/icons/up-right-and-down-left-from-center.svg" className="inline-icon" alt="Full screen icon" /> **Full screen** in the top right.

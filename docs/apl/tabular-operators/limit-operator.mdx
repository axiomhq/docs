# limit operator

Returns up to the specified number of rows.

## Syntax

```apl copy
| limit
```

The [take operator](/docs/apl/tabular-operators/take-operator) is an alias to the Limit operator. 

## Arguments

| **name**     | **type**  | **description**                                                       |
| ------------ | --------- | --------------------------------------------------------------------- |
| Number Of Rows | **integer** | The number of rows to return. You can specify any numeric expression. |

## Example

```apl copy
['http-logs']
| limit 10000
```

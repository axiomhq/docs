## Configure HTTP destination

Configure the following:

- In **URL**, enter the path to the HTTP destination where you want to route data.
- In **Format**, specify the format in which Axiom sends data to the destination. For example, CSV or JSON.
- Optional: In **Headers**, specify any headers you want Axiom to send to the destination.
- In **Authorization**, choose how you want to authorize the request. After choosing the desired authorization type, provide any additional credentials required.

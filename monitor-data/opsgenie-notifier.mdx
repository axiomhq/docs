Configure an Opsgenie notifier to integrate Axiom with Opsgenie's incident management features.

To create an Opsgenie notifier:

1. Create an API integration in Opsgenie. For more information, see the [Opsgenie documentation](https://support.atlassian.com/opsgenie/docs/create-a-default-api-integration/).
2. Click the **Monitors** tab, then click **Manage notifiers**.
3. Click **New notifier**.
4. Enter a name for your notifier.
5. Click **Opsgenie**.
6. Enter your generated API key.
7. Select your Opsgenie instance region.
8. Click **Create**.

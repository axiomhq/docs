<Warning>
Use this endpoint to query data within a specific edge deployment. Query results are returned directly from the edge without routing through other infrastructure. For more information, see [Query data](/restapi/query) and [Edge deployments](/reference/edge-deployments).

The base domain for this endpoint is the base domain of the edge deployment you want to query. For a list of base domains, see [Edge deployments](/reference/edge-deployments).
</Warning>

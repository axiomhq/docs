<Warning>
The base domain for this endpoint is `https://api.axiom.co`, irrespective of your edge deployment.

To ingest data to your edge deployment, use the [Ingest data to edge deployment](/restapi/endpoints/ingestToDataset) endpoint. For more information, see [Ingest data](/restapi/ingest) and [Edge deployments](/reference/edge-deployments).
</Warning>

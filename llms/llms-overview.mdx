## AI chat

To get an answer about Axiom from the documentation’s built-in AI assistant, press <kbd>Cmd/Ctrl</kbd>&nbsp;<kbd>K</kbd>, and then type your question.

## Interpret individual pages

To interpret individual pages of the documentation with an LLM, click <Icon icon="chevron-down" iconType="solid" /> in the top right, and then choose one of the following options:
- Copy the page in Markdown format and manually pass it to LLMs
- View the page in Markdown format
- Open the page in ChatGPT
- Open the page in Claude

## Interpret the full docs

To make it easy for LLMs to interpret the documentation, Axiom offers the following based on the [/llms.txt standard](https://llmstxt.org/):

- [List of docs pages](https://axiom.co/docs/llms.txt)
- [Full documentation](https://axiom.co/docs/llms-full.txt)
- [List of query reference pages](/llms-apl.md)

Access one of the above documents and pass it to your LLM to start interpreting the Axiom documentation.

## Use Axiom docs with AI agents

To help AI agents understand Axiom's capabilities, Axiom offers a [capability summary](/skill.md) that follows the [skill.md specification](https://agentskills.io/specification). It's a structured, machine-readable format that makes Axiom's capabilities, required inputs, and constraints explicit so that AI agents can use them reliably.

<Tip>
`llms.txt` and `skill.md` complement each other:
- `llms.txt` acts as a table of contents, mapping every documentation page so agents can locate specific information.
- `skill.md` provides a high-level overview of what Axiom offers, including key actions, expected inputs, and known constraints.
</Tip>

### Use skill.md with agents

Add Axiom's capabilities to an AI agent's context with the [skills CLI](https://www.npmjs.com/package/skills):

```bash
npx skills add https://axiom.co/docs
```

### Discovery endpoints

AI agents can discover and fetch skill files programmatically through the following endpoints:

- `GET https://axiom.co/docs/.well-known/agent-skills/index.json` returns a JSON manifest listing all available skills. This follows the [agent-skills 0.2.0 discovery spec](https://schemas.agentskills.io/discovery/0.2.0/schema.json).
- `GET https://axiom.co/docs/.well-known/skills/index.json` returns a JSON manifest in the original discovery format.

Use SQL functions to parse and reconstruct SQL statements stored as strings in your datasets. They're useful in database monitoring, audit log analysis, and SQL validation pipelines.

## List of functions

| Function | Description |
| -------- | ----------- |
| [parse_sql](/apl/scalar-functions/sql-functions/parse-sql) | Parses a SQL statement string into a structured dictionary of its components. |
| [format_sql](/apl/scalar-functions/sql-functions/format-sql) | Converts the structured dictionary produced by `parse_sql` back into a SQL string. |
